# Snake Game using Python and Pygame

This is a simple Snake Game developed using Python and the Pygame library.

## Features
- Snake moves using arrow keys
- Food generation system
- Score tracking
- Game over on wall or self collision

## Requirements
- Python 3.x
- Pygame

## How to Run
1. Install pygame:
   pip install pygame
2. Run the game:
   python snake_game.py
